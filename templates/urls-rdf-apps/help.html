Projects

  • BIBCAT an open-source project sponsored by KnowledgesLink LLC and is main component of the Catalog Pull Platform. BIBCAT provides a core set of RDF-based utilities for managing bibliographic based RDF graphs using multiple vocabularies with a focus on BIBFRAME and schema.org
  • The Colorado Alliance of Research Libraries's BIBCAT Pilot is an Library Linked-Data publishing project for transforming MARC XML from the Alliance's Gold Rush Library Comparison Systems to BIBFRAME RDF that is then published to the web in schema.org metadata for indexing by commercial search engines like Google and Bing. All open-source code is hosted on Github at https://github.com/KnowledgeLinks/alliance-bibcat
  • DPLA Service Hub is the open-source software behind the Plains2Peaks.org DP.LA regional service hub for Colorado and Wyoming.
  • The last two projects are an open-source thesis self-submission application for Colorado College's Seniors to upload their thesis and supporting datasets to Colorado College's digital repository. The Colorado College institutional knowledge graph to support this application is hosted on Github at https://github.com/Tutt-Library/tiger-catalog.
